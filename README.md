# Hybrid Vulnerability Scanner

## Overview

A Python-based cybersecurity tool that performs:

- Network Port Scanning
- Service Detection
- Web Security Header Analysis
- HTTPS Verification
- Directory Discovery
- Risk Assessment
- TXT Report Generation
- PDF Report Generation

## Technologies Used

- Python
- Nmap
- Requests
- ReportLab

## Installation

```bash
pip install -r requirements.txt
```
